Data Science with R - A set of examples

Welcome to the githib resource containing interesting examples of applying R to a range of data science problems, including:

  • Text Mining, with an example that creates a wordcloud from the opening chapter of Ulysses (Episode 1 - Telemachus) by James Joyce. Code based on the nice example of wordclouds in R already published on the web, and also using the available online text for Ulysses. For more information on text mining, see wikepedia and the tm package in R
